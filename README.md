# Restaurant Kiosk

간단한 식당 키오스크 웹 애플리케이션입니다.

## Features

* 메뉴 선택
* 수량 증가 / 감소
* 총 주문 금액 계산
* 주문 버튼
* 반응형 디자인

## Tech Stack

* HTML
* CSS
* JavaScript
